# 예시 입력

## 간단 요청
```
"Google OAuth 로그인 기능 이슈 만들어줘"
```

AI가 자동으로:
1. 마일스톤 `RES-02 Google OAuth 로그인` 생성
2. 서브이슈 3~5개 생성 (`[RES-02-1]`, `[RES-02-2]`, ...)

## 상세 요청
```
Google OAuth 로그인 기능 필요해.
- Google Cloud Console 설정
- OAuth 엔드포인트 구현
- 사용자 정보 DB 저장
- JWT 토큰 발급
담당자는 yuunicode, 라벨은 feature로 해줘.
```

## 생성 결과 예시

### 마일스톤
`RES-02 Google OAuth 로그인`

### 서브이슈
| 이슈 | 타이틀 |
|------|--------|
| #2 | [RES-02-1] Google OAuth 클라이언트 설정 |
| #3 | [RES-02-2] OAuth 엔드포인트 구현 |
| #4 | [RES-02-3] 사용자 정보 DB 저장 |
| #5 | [RES-02-4] JWT 토큰 및 세션 관리 |
